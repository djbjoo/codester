# Markdown
- standard for documentation in coding world
- text-based
- open standard

## Formatting commands
- headings start with a hash/pound
- bullets are dashes or asterisks
- underscore is _italic_
- bold is two **asterisks**
- more hashes = smaller headings (level 1 or 2 headings)
